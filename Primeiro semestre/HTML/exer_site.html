Exercício inicial HTML


Características do HTML

As principais características do HTML são:

  • Documentos HTML são arquivos escritos em ASCII - texto;
  • Podem ser criados em qualquer editor de texto (visual studio, emacs, notepad, etc.);
  • Existem editores específicos para várias plataformas;
  • Existem conversores de vários formatos, por exemplo, doc para html;
  • A unidade mínima de informação é a “página”.

Marcações/Tags HTML

O que são as marcações/tags?

As marcações do HTML - tags - consistem do sinal (<), seguida pelo nome da marcação ("p" para parágrafo, "title" para titúlo e etc) e fechada por (>). De um modo geral, as tags aparecem em pares, por exemplo, <h1> Cabeçalho </h1>. O símbolo que termina uma determinada marcação é igual aquele que a inicia, antecedido por uma barra (/) e precedido pelo texto referente.

Exemplos de tags:

  • <title> para títulos;
  • <h1>, <h2> - <h6> para aparência de cabeçalho;
  • <p> para um parágrafo;
  • <ul> para lista desordenada;
  • <ol> para lista ordenada;
  • <li> para criar um item na lista;
  • <b> para transformar o texto em negrito
  • <div> para criar uma divisão dentro da página;
  • <table> para criar uma tabela;
  • <textarea> para determinar uma caixa de texto.

Estrutura HTML

A estrutura html é definida do seguinte modo:

<!DOCTYPE html> <head> -- Aqui é onde fica o cabeçalho, os links de outros arquivos, título da página e etc. -- <meta charset="UTF-8"> <title>Título</title> </head> <body> -- Aqui no body é onde fica as divs, e a estrutura da página em si. -- </body> </html>


Marcadores de estilo

A tag <style> é um marcador de estilo, usado para aplicar CSS no elemento desejado, ou seja, para customizá-lo e deixá-lo do jeito que o usuário deseja visualmente. Por exemplo: utilizei a tag style do seguinte modo para aplicar CSS logo acima, no tópico "Estrutura HTML", e colocarei abaixo também:

<div style="background-color: #e6e6e6 ; border-left-style: solid; border-color: rgb(146, 0, 0); border-width: 7px; width: 70%">

Cores de texto e padrões de fundo

As cores de texto e os padrões de fundo são definidos através da tag <style>, com CSS. A cor de texto é definido por "color: [cor desejada]" e o padrão de fundo é definido por "background-color: [cor desejada]", caso queira uma cor, "background-image: url(caminho da imagem desejada)", adicionando "background-repeat:", seguido de "repeat", "repeat-x", "repeat-y" ou "no-repeat" para diferentes modos de repetição da imagem.


Imagens

Podemos também adicionar imagens no nosso site através da tag:

<img src="[caminho da imagem]" alt="[nome alternativo para caso a imagem não possa ser exibida]">



Links

Um link pode ser adicionado à uma palavra, botão ou imagem da seguinte forma:

<a href="[link da página]">[texto, imagem ou botão para levar até a página]</a>

Tabelas

No HTML podemos fazer tabelas também, com vários usos, dependendo que o usário deseja. Podemos criar uma tabela do seguinte modo:

<table border="1"> <tr> <th>Produto</th> <th>Valor</th> <th>Quantidade</th> </tr> <tr> <td>Placa de Vídeo Asus NVIDIA GeForce RTX 3090</td> <td>R$ 24.439,90</td> <td>1</td> </tr> <tr> <td>Headset Gamer Redragon Pandora</td> <td>R$ 259,90</td> <td>2</td> </tr> <tr> <td>Memória HyperX Fury, 8GB, 2666MHz</td> <td>R$ 411,65</td> <td>2</td> </tr> <tr> <td>Fonte Redragon RGPS, 700W</td> <td>R$ 603,42</td> <td>1</td> </tr> </table>

Onde:

  • border = estilo da borda;
  • tr = nova linha da tabela;
  • td = nova coluna da tabela.

Criando a tabela, ela fica assim:

Produto Valor Quantidade
Placa de Vídeo Asus NVIDIA GeForce RTX 3090 R$ 24.439,90 1
Headset Gamer Redragon Pandora R$ 259,90 2
Memória HyperX Fury, 8GB, 2666MHz R$ 411,65 2
Fonte Redragon RGPS, 700W R$ 603,42 1

Adicionando CSS à tabela, ela fica assim:

Formulários

aassA melhor função para uma página da web coletar informações do usuário é através de um formulário. Para fazê-lo, é simples, como no exemplo à seguir:

<form> <label for="user">Usuário:</label><br> <input type="text" id="user" name="user"><br> <label for="password">Senha:</label><br> <input type="password" id="password" name="password"> <input type="submit" value="Enviar"> </form>
  • Em label, definimos o rótulo que fica acima da caixa de texto do formulário;
  • No type do input, colocamos o tipo de input que desejamos colocar. No exemplo, temos o primeiro input como text e o segundo como password;
  • No último input, colocamos o tipo como submit, para se referir ao botão que envia as informações digitadas pelo usuário.
